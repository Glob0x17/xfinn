xfinn - Jellyfin Client for Apple TV

Getting Started

xfinn is a native tvOS client for Jellyfin media servers. To use the app, you need:

  • A running Jellyfin server (self-hosted or remote)
  • An Apple TV 4K (recommended) or Apple TV HD
  • Your Jellyfin server URL and login credentials

Frequently Asked Questions

How do I connect to my Jellyfin server?

Launch xfinn and enter your Jellyfin server URL (e.g., http://192.168.1.100:8096 or https://jellyfin.example.com). Then log in with your Jellyfin username and password.

The app can't find my server on the local network

Make sure your Apple TV and Jellyfin server are on the same network. Check that you've granted local network access permission to xfinn in Settings > Privacy > Local Network.

Video playback is buffering or stuttering

Try lowering the video quality in the playback settings. If your server is transcoding, ensure it has enough processing power. For best results, use media files that can be Direct Played.

Subtitles are not showing correctly

xfinn supports multiple subtitle formats. If subtitles aren't appearing, try selecting a different subtitle track in the playback controls. SRT and ASS formats work best.

How do I change the video quality?

During playback, press the touchpad on your Apple TV remote to access playback controls. Select the quality option to choose your preferred resolution (Auto, 4K, 1080p, 720p, etc.).

Can I use xfinn with multiple Jellyfin servers?

Currently, xfinn connects to one server at a time. You can log out and connect to a different server from the settings menu.

Troubleshooting

Connection Issues

  • Verify your Jellyfin server is running and accessible
  • Check your server URL includes the correct port (default: 8096)
  • Ensure your firewall allows connections on the Jellyfin port
  • For remote access, make sure your server is properly configured for external connections

Playback Issues

  • Restart the app if playback becomes unresponsive
  • Check your network connection speed
  • Verify your Jellyfin server has the required codecs for transcoding
  • Try a different video file to isolate the issue

Need More Help?

If you're experiencing issues not covered here, please reach out:

  • Check the Jellyfin documentation at jellyfin.org/docs
  • Join the Jellyfin community for general media server support
  • Report app-specific bugs through the feedback option
